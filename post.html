Private Cloud Server

This project was a part of my masters program in Software project management.
We worked in a Team of 3 but everybody tried to work differently so that everybody gets a hands on knowledgemodule.

The private cloud was hosted on a powerful UNIX Engine called FreeBSD. It is also used by various cloud services, for eg. Netflix and iCloud.

We used a open source API called Nextcloud, various prerequisites were needed before installation. Details on prerequisites can be found on Nextcloud Documentation.

To access the webversion a DNS is needed. DynDNS provider DuckDNS was used to access the nextcloud via the internet. It can be setup using the home router.

We worked on this project for 6 weeks.

Technologies:
  • - FreeBSD
  • - MySQL
  • - Apache Webserver
  • - DuckDNS Domain
  • - Raspberry Pi3
  • - Old unused Laptop
  • - PHP
