Private Cloud Server

This project was a part of my masters program in Software project management.
We worked in a Team of 3 but everybody tried to work differently so that everybody gets a hands on knowledgemodule.

The private cloud was hosted on a powerful UNIX Engine called FreeBSD. It is also used by various cloud services, for eg. Netflix and iCloud.

We used a open source API called Nextcloud, various prerequisites were needed before installation. Details on prerequisites can be found on Nextcloud Documentation.

To access the webversion a DNS is needed. DynDNS provider DuckDNS was used to access the nextcloud via the internet. It can be setup using the home router.

We worked on this project for 6 weeks.

Coherent heterodyne optical transmission system

In this project, coherent detection with heterodyne technique and few DSP techniques such as digital predistortion and carrier phase estimation for efficient transmissions of data is presented and characterized. In coherent detection, an optical receiver can monitor the phase of an optical transmitter to obtain any phase and frequency information carried by a transmitted signal. The thesis is also centered on developing Python script files for simulation of the optical communication system, where the development of drivers for test and measuring equipmentâ€™s for signal generation, acquisition, and transmission are conducted. The main objective of this thesis is to design, implement and test an optical coherent heterodyne transceiver.
In this project following goals are targeted to be accomplished:


  • - Describe and study the fundamental principles of coherent communication.

  • - Review and design the optical coherent communication system and the architecture used to implement heterodyne detection technique.

  • - Design of an experimental set-up to incorporate an optical coherent communication system using commercially available devices.

  • - Design and performance analysis of the various compensation techniques required in a true, coherent optical receiver such as clock recovery, frequency offset estimation and phase estimation.

  • - Test and characterize the implemented communication system.

Signal processing techniques on Matlab

In this computational exercise, the fundamentals of a simple radio system are developed with diversity and space multiplex options using a numerical computer algebra system. For this purpose, the individual components of the system, the transmitter, a radio channel and a receiver, are simulated on the computer. As simulation environment MATLAB was be used. Developing a biodegradable excrement packets

A bio-degradable packet idea was discussed in order to solve feces problem in base camp of Mt. Everest. Every year many climbers, explorers, including amateur tourists climb Mt.Everest. Through decades of use and exploration the area is extremely polluted by left excrements. To solve this problem we have designed a biodegradable pack(storage packet) where one can store their excrements and dump it in the recycling facility.
The details are in this presentation. We are still working to make this idea to come into life.
